\doxysubsection{Detailed Description}
Each JS task object created via {\ttfamily console.\+create\+Task} has a corresponding {\ttfamily \mbox{\hyperlink{classv8__inspector_1_1TaskInfo}{Task\+Info}}} object on the C++ side (in a 1\+:1 relationship).

The {\ttfamily \mbox{\hyperlink{classv8__inspector_1_1TaskInfo}{Task\+Info}}} holds on weakly to the JS task object. The JS task objects uses a private symbol to store a pointer to the {\ttfamily \mbox{\hyperlink{classv8__inspector_1_1TaskInfo}{Task\+Info}}} object (via \mbox{\hyperlink{classv8_1_1External}{v8\+::\+External}}).

The {\ttfamily \mbox{\hyperlink{classv8__inspector_1_1TaskInfo}{Task\+Info}}} objects holds all the necessary information we need to properly cancel the corresponding async task then the JS task object gets GC\textquotesingle{}ed.

\doxysubsection{Detailed Description}
Disallows garbage collection finalizations. 

Any garbage collection triggers result in a crash when in this scope.

Note that the garbage collector already covers paths that can lead to garbage collections, so user code does not require checking {\ttfamily \mbox{\hyperlink{classcppgc_1_1subtle_1_1DisallowGarbageCollectionScope_a54a8af5c1808eb316f5b6e1602c3ac8c}{Is\+Garbage\+Collection\+Allowed()}}} before allocations.

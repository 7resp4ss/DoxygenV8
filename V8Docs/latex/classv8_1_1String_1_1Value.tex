\doxysubsection{Detailed Description}
Converts an object to a two-\/byte (UTF-\/16-\/encoded) string.

If conversion to a string fails (eg. due to an exception in the to\+String() method of the object) then the \mbox{\hyperlink{classv8_1_1String_1_1Value_a6cf7c33188e79b426dc7e132a8767520}{length()}} method returns 0 and the $\ast$ operator returns NULL.

WARNING\+: This will unconditionally copy the contents of the Java\+Script string, and should be avoided in situations where performance is a concern.

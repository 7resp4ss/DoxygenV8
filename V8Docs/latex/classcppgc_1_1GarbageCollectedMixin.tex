\doxysubsection{Detailed Description}
Base class for managed mixin objects. 

Such objects cannot be constructed directly but must be mixed into the inheritance hierarchy of a \mbox{\hyperlink{classcppgc_1_1GarbageCollected}{Garbage\+Collected}} object.

Types inheriting from \mbox{\hyperlink{classcppgc_1_1GarbageCollectedMixin}{Garbage\+Collected\+Mixin}} must override a virtual method of signature {\ttfamily void \mbox{\hyperlink{classcppgc_1_1GarbageCollectedMixin_a91039d18f0f1d66c08f6f9546f6c268b}{Trace(cppgc\+::\+Visitor$\ast$) const}}} that dispatchs all managed pointers to the visitor and delegates to base classes.

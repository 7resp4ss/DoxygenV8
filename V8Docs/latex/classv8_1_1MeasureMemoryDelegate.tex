\doxysubsection{Detailed Description}
The delegate is used in \mbox{\hyperlink{classv8_1_1Isolate_a1393298ca557f6cbc312116d77c9b501}{Isolate\+::\+Measure\+Memory}} API.

It specifies the contexts that need to be measured and gets called when the measurement is completed to report the results.

Both \mbox{\hyperlink{classv8_1_1MeasureMemoryDelegate_aba994ca37dd3f851035e0830a0dd3ab1}{Measurement\+Complete()}} callbacks will be invoked on completion. Each implementation of this class should hence implement only one of them, and leave the other empty.

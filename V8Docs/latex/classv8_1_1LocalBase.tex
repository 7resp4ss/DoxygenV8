\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class v8\+::\+Local\+Base$<$ T $>$}

A base class for local handles. 

Its implementation depends on whether direct handle support is enabled. When it is, a local handle contains a direct pointer to the referenced object, otherwise it contains an indirect pointer.

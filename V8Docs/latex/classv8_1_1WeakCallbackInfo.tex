When a weak callback is first invoked the embedders {\itshape must} Reset() the handle which triggered the callback. The handle itself is unusable for anything else. No other \mbox{\hyperlink{classv8_1_1V8}{V8}} API calls may be called in the first callback. Additional work requires scheduling a second invocation via {\ttfamily \mbox{\hyperlink{classv8_1_1WeakCallbackInfo_af0f671ce037ff91319683e8f01645865}{Set\+Second\+Pass\+Callback()}}} which will be called some time after all the initial callbacks are processed.

The second pass callback is prohibited from executing Java\+Script. Embedders should schedule another callback in case this is required.

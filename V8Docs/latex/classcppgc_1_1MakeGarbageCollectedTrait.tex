\subsubsection*{template$<$typename T$>$\newline
class cppgc\+::\+Make\+Garbage\+Collected\+Trait$<$ T $>$}

Default trait class that specifies how to construct an object of type T. Advanced users may override how an object is constructed using the utilities that are provided through \mbox{\hyperlink{classcppgc_1_1MakeGarbageCollectedTraitBase}{Make\+Garbage\+Collected\+Trait\+Base}}.

Any trait overriding construction must
\begin{DoxyItemize}
\item allocate through {\ttfamily \mbox{\hyperlink{classcppgc_1_1MakeGarbageCollectedTraitBase}{Make\+Garbage\+Collected\+Trait\+Base}}$<$T$>$\+::\+Allocate};
\item mark the object as fully constructed using {\ttfamily \mbox{\hyperlink{classcppgc_1_1MakeGarbageCollectedTraitBase}{Make\+Garbage\+Collected\+Trait\+Base}}$<$T$>$\+::\+Mark\+Object\+As\+Fully\+Constructed}; 
\end{DoxyItemize}

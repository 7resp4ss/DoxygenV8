\subsubsection*{template$<$typename T$>$\newline
class cppgc\+::\+Make\+Garbage\+Collected\+Trait$<$ T $>$}

Default trait class that specifies how to construct an object of type \mbox{\hyperlink{classT}{T}}. 

Advanced users may override how an object is constructed using the utilities that are provided through \mbox{\hyperlink{classcppgc_1_1MakeGarbageCollectedTraitBase}{Make\+Garbage\+Collected\+Trait\+Base}}.

Any trait overriding construction must

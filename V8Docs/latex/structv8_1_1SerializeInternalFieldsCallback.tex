\doxysubsection{Detailed Description}
Callback and supporting data used in \mbox{\hyperlink{classv8_1_1SnapshotCreator}{Snapshot\+Creator}} to implement embedder logic to serialize internal fields of v8\+::\+Objects. Internal fields that directly reference \mbox{\hyperlink{classv8_1_1V8}{V8}} objects are serialized without calling this callback. Internal fields that contain aligned pointers are serialized by this callback if it returns non-\/zero result. Otherwise it is serialized verbatim.

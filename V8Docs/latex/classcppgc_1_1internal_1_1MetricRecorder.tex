\doxysubsection{Detailed Description}
Base class used for reporting GC statistics histograms. 

Embedders interested in collecting histograms should implement the virtual Add\+Main\+Thread\+Event methods below and pass an instance of the implementation during \mbox{\hyperlink{classcppgc_1_1internal_1_1Heap}{Heap}} creation.

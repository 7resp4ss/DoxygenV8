\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classv8_1_1V8}{V8}} interface for Web\+Assembly streaming compilation. When streaming compilation is initiated, \mbox{\hyperlink{classv8_1_1V8}{V8}} passes a \{\mbox{\hyperlink{classv8_1_1WasmStreaming}{Wasm\+Streaming}}\} object to the embedder such that the embedder can pass the input bytes for streaming compilation to \mbox{\hyperlink{classv8_1_1V8}{V8}}.

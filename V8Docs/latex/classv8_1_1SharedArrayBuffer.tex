Create a new \mbox{\hyperlink{classv8_1_1SharedArrayBuffer}{Shared\+Array\+Buffer}} with an existing backing store. The created array keeps a reference to the backing store until the array is garbage collected. Note that the Is\+External bit does not affect this reference from the array to the backing store.

In future Is\+External bit will be removed. Until then the bit is set as follows. If the backing store does not own the underlying buffer, then the array is created in externalized state.

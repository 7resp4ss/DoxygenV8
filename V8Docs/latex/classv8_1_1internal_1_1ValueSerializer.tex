\doxysubsection{Detailed Description}
Writes \mbox{\hyperlink{classv8_1_1internal_1_1V8}{V8}} objects in a binary format that allows the objects to be cloned according to the HTML structured clone algorithm.

Format is based on Blink\textquotesingle{}s previous serialization logic.

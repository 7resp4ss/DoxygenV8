Contains the import attributes for this request in the form\+: \mbox{[}key1, value1, source\+\_\+offset1, key2, value2, source\+\_\+offset2, ...\mbox{]}. 

The keys and values are of type \mbox{\hyperlink{classv8_1_1String}{v8\+::\+String}}, and the source offsets are of type \mbox{\hyperlink{classv8_1_1Int32}{Int32}}. Use \mbox{\hyperlink{classv8_1_1Module_a17b96260eb80ec329df6c059bec82675}{Module\+::\+Source\+Offset\+To\+Location}} to convert the source offsets to Locations with line/column numbers.

All attributes present in the module request will be supplied in this list, regardless of whether they are supported by the host. Per \href{https://tc39.es/proposal-import-attributes/\#sec-hostgetsupportedimportattributes}{\texttt{ https\+://tc39.\+es/proposal-\/import-\/attributes/\#sec-\/hostgetsupportedimportattributes}}, hosts are expected to throw for attributes that they do not support (as opposed to, for example, ignoring them).

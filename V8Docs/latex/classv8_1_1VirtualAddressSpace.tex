\doxysubsection{Detailed Description}
Class to manage a virtual memory address space.

This class represents a contiguous region of virtual address space in which sub-\/spaces and (private or shared) memory pages can be allocated, freed, and modified. This interface is meant to eventually replace the \mbox{\hyperlink{classv8_1_1PageAllocator}{Page\+Allocator}} interface, and can be used as an alternative in the meantime.

This API is not yet stable and may change without notice!

The granularity of page allocations and, by extension, of subspace allocations. This is guaranteed to be a power of two and a multiple of the \mbox{\hyperlink{classv8_1_1VirtualAddressSpace_afd51bd33692e85fa4742f62c9c832ffe}{page\+\_\+size()}}. In practice, this is equal to the page size on most OSes, but on Windows it is usually 64KB, while the page size is 4KB.

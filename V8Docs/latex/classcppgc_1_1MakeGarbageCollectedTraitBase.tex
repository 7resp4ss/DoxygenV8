\subsubsection*{template$<$typename T$>$\newline
class cppgc\+::\+Make\+Garbage\+Collected\+Trait\+Base$<$ T $>$}

Base trait that provides utilities for advancers users that have custom allocation needs (e.\+g., overriding size). It\textquotesingle{}s expected that users override \mbox{\hyperlink{classcppgc_1_1MakeGarbageCollectedTrait}{Make\+Garbage\+Collected\+Trait}} (see below) and inherit from \mbox{\hyperlink{classcppgc_1_1MakeGarbageCollectedTraitBase}{Make\+Garbage\+Collected\+Trait\+Base}} and make use of the low-\/level primitives offered to allocate and construct an object.

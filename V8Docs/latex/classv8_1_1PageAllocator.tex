Can be implemented by an embedder to manage large host OS allocations.

{\footnotesize\ttfamily virtual bool v8\+::\+Page\+Allocator\+::\+Decommit\+Pages (\begin{DoxyParamCaption}\item[{void $\ast$}]{address,  }\item[{size\+\_\+t}]{size }\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [pure virtual]}}

Decommits any wired memory pages in the given range, allowing the OS to reclaim them, and marks the region as inacessible (k\+No\+Access). The address range stays reserved and can be accessed again later by changing its permissions. However, in that case the memory content is guaranteed to be zero-\/initialized again. The memory must have been previously allocated by a call to Allocate\+Pages. Returns true on success, false otherwise.

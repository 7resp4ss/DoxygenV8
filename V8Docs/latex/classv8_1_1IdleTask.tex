\doxysubsection{Detailed Description}
An \mbox{\hyperlink{classv8_1_1IdleTask}{Idle\+Task}} represents a unit of work to be performed in idle time. 

The Run method is invoked with an argument that specifies the deadline in seconds returned by Monotonically\+Increasing\+Time(). The idle task is expected to complete by this deadline.

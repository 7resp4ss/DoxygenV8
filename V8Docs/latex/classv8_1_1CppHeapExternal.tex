\doxysubsection{Detailed Description}
A Java\+Script value that wraps a {\ttfamily \mbox{\hyperlink{classcppgc_1_1GarbageCollected}{cppgc\+::\+Garbage\+Collected}}$<$T$>$} object allocated on the managed C++ heap (\mbox{\hyperlink{classv8_1_1CppHeap}{Cpp\+Heap}}). This type of value is mainly used to associate C++ data structures which aren\textquotesingle{}t exposed to Java\+Script with Java\+Script objects.

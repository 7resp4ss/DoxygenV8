\doxysubsection{Detailed Description}
An array to hold \mbox{\hyperlink{classv8_1_1Primitive}{Primitive}} values. 

This is used by the embedder to pass host defined options to the Script\+Options during compilation.

This is passed back to the embedder as part of Host\+Import\+Module\+Dynamically\+Callback for module loading.

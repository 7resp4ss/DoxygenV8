\mbox{\hyperlink{classv8_1_1internal_1_1ConcurrentMarking_1_1JobTaskMinor_af5e17773a5bba22169f55c790e4a2d22}{Run()}} is only invoked if the number of threads previously running \mbox{\hyperlink{classv8_1_1internal_1_1ConcurrentMarking_1_1JobTaskMinor_af5e17773a5bba22169f55c790e4a2d22}{Run()}} was less than the value returned. In general, this should return the latest number of incomplete work items (smallest unit of work) left to process, including items that are currently in progress. $\vert$worker\+\_\+count$\vert$ is the number of threads currently assigned to this job which some callers may need to determine their return value.

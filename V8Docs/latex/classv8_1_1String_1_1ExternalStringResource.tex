\doxysubsection{Detailed Description}
An \mbox{\hyperlink{classv8_1_1String_1_1ExternalStringResource}{External\+String\+Resource}} is a wrapper around a two-\/byte string buffer that resides outside \mbox{\hyperlink{classv8_1_1V8}{V8}}\textquotesingle{}s heap. 

Implement an \mbox{\hyperlink{classv8_1_1String_1_1ExternalStringResource}{External\+String\+Resource}} to manage the life cycle of the underlying buffer. Note that the string data must be immutable.

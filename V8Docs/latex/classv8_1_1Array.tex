{\footnotesize\ttfamily \mbox{\hyperlink{classv8_1_1Maybe}{Maybe}}$<$ void $>$ v8\+::\+Array\+::\+Iterate (\begin{DoxyParamCaption}\item[{\mbox{\hyperlink{classv8_1_1Local}{Local}}$<$ \mbox{\hyperlink{classv8_1_1Context}{Context}} $>$}]{context,  }\item[{\mbox{\hyperlink{classv8_1_1Array_adfd46ea5e610d3511ea6013f2d15df44}{Iteration\+Callback}}}]{callback,  }\item[{void $\ast$}]{callback\+\_\+data }\end{DoxyParamCaption})}



Calls \{callback\} for every element of this array, passing \{callback\+\_\+data\} as its \{data\} parameter. 

This function will typically be faster than calling \{\mbox{\hyperlink{classv8_1_1Object_abe2967d3b14fe56c00718cc29441d67b}{Get()}}\} repeatedly. As a consequence of being optimized for low overhead, the provided callback must adhere to the following restrictions\+:
\begin{DoxyItemize}
\item It must not allocate any \mbox{\hyperlink{classv8_1_1V8}{V8}} objects and continue iterating; it may allocate (e.\+g. an error message/object) and then immediately terminate the iteration.
\item It must not modify the array being iterated.
\item It must not call back into \mbox{\hyperlink{classv8_1_1V8}{V8}} (unless it can guarantee that such a call does not violate the above restrictions, which is difficult).
\item The \{Local$<$\+Value$>$ element\} must not \char`\"{}escape\char`\"{}, i.\+e. must not be assigned to any other \{\mbox{\hyperlink{classv8_1_1Local}{Local}}\}. Creating a \{\mbox{\hyperlink{classv8_1_1Global}{Global}}\} from it, or updating a \mbox{\hyperlink{classv8_1_1TypecheckWitness}{v8\+::\+Typecheck\+Witness}} with it, is safe. These restrictions may be lifted in the future if use cases arise that justify a slower but more robust implementation.
\end{DoxyItemize}

Returns \{Nothing\} on exception; use a \{\mbox{\hyperlink{classv8_1_1TryCatch}{Try\+Catch}}\} to catch and handle this exception. When the \{callback\} returns \{k\+Exception\}, iteration is terminated immediately, returning \{Nothing\}. By returning \{k\+Break\}, the callback can request non-\/exceptional early termination of the iteration.

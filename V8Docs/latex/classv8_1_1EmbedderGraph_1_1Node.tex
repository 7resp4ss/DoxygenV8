Returns the address of the object in the embedder heap, or nullptr to not specify the address. 

If this address is provided, then \mbox{\hyperlink{classv8_1_1V8}{V8}} can generate consistent IDs for objects across subsequent heap snapshots, which allows devtools to determine which objects were retained from one snapshot to the next. This value is used only if Get\+Native\+Object returns nullptr.

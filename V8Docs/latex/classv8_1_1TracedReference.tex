\subsubsection*{template$<$typename T$>$\newline
class v8\+::\+Traced\+Reference$<$ T $>$}

A traced handle without destructor that clears the handle. The embedder needs to ensure that the handle is not accessed once the \mbox{\hyperlink{classv8_1_1V8}{V8}} object has been reclaimed. For more details see \mbox{\hyperlink{classv8_1_1BasicTracedReference}{Basic\+Traced\+Reference}}.

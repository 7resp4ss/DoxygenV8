Used in combination with $\vert$\+Is\+Root$\vert$. 

Called by \mbox{\hyperlink{classv8_1_1V8}{V8}} when an object that is backed by a handle is reclaimed by a non-\/tracing garbage collection. It is up to the embedder to reset the original handle.

Note that the $\vert$handle$\vert$ is different from the handle that the embedder holds for retaining the object. It is up to the embedder to find the original handle via the object or class id. \mbox{\Hypertarget{classv8_1_1EmbedderRootsHandler_a9c8ce618031aaea77c67a8387e6ea5a3}\label{classv8_1_1EmbedderRootsHandler_a9c8ce618031aaea77c67a8387e6ea5a3}}

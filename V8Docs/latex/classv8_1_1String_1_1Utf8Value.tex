\doxysubsection{Detailed Description}
Converts an object to a null-\/terminated UTF-\/8-\/encoded character array. Useful if you want to print the object. If conversion to a string fails (e.\+g. due to an exception in the to\+String() method of the object) then the \mbox{\hyperlink{classv8_1_1String_1_1Utf8Value_acfaec2ea68bce4b40cab2f77692e759b}{length()}} method returns 0 and the $\ast$ operator returns NULL.

WARNING\+: This will unconditionally copy the contents of the Java\+Script string, and should be avoided in situations where performance is a concern. Consider using \mbox{\hyperlink{classv8_1_1String_a7a15f155614bef1fdda4dfc1072dc846}{Write\+Utf8()}} instead.

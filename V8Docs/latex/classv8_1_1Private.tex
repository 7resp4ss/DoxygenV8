Retrieve a global private symbol. 

If a symbol with this name has not been retrieved in the same isolate before, it is created. Note that private symbols created this way are never collected, so they should only be used for statically fixed properties. Also, there is only one global name space for the names used as keys. To minimize the potential for clashes, use qualified names as keys, e.\+g., \char`\"{}\+Class\#property\char`\"{}.

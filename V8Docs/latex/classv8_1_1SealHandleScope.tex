\doxysubsection{Detailed Description}
A \mbox{\hyperlink{classv8_1_1SealHandleScope}{Seal\+Handle\+Scope}} acts like a handle scope in which no handle allocations are allowed. 

It can be useful for debugging handle leaks. Handles can be allocated within inner normal Handle\+Scopes.

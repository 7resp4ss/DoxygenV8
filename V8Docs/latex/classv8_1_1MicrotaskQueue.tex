\doxysubsection{Detailed Description}
Represents the microtask queue, where microtasks are stored and processed. \href{https://html.spec.whatwg.org/multipage/webappapis.html\#microtask-queue}{\texttt{ https\+://html.\+spec.\+whatwg.\+org/multipage/webappapis.\+html\#microtask-\/queue}} \href{https://html.spec.whatwg.org/multipage/webappapis.html\#enqueuejob(queuename,-job,-arguments)}{\texttt{ https\+://html.\+spec.\+whatwg.\+org/multipage/webappapis.\+html\#enqueuejob(queuename,-\/job,-\/arguments)}} \href{https://html.spec.whatwg.org/multipage/webappapis.html\#perform-a-microtask-checkpoint}{\texttt{ https\+://html.\+spec.\+whatwg.\+org/multipage/webappapis.\+html\#perform-\/a-\/microtask-\/checkpoint}}

A \mbox{\hyperlink{classv8_1_1MicrotaskQueue}{Microtask\+Queue}} instance may be associated to multiple Contexts by passing it to \mbox{\hyperlink{classv8_1_1Context_a11648311890e6127d92335fc438b8211}{Context\+::\+New()}}, and they can be detached by \mbox{\hyperlink{classv8_1_1Context_a841c7dd92eb8c57df92a268a164dea97}{Context\+::\+Detach\+Global()}}. The embedder must keep the \mbox{\hyperlink{classv8_1_1MicrotaskQueue}{Microtask\+Queue}} instance alive until all associated Contexts are gone or detached.

Use the same instance of \mbox{\hyperlink{classv8_1_1MicrotaskQueue}{Microtask\+Queue}} for all Contexts that may access each other synchronously. E.\+g. for Web embedding, use the same instance for all origins that share the same URL scheme and e\+TLD+1.

Executing scripts inside the callback will not re-\/trigger microtasks and the callback.

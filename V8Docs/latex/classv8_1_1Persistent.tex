\doxysubsection{Detailed Description}
\subsubsection*{template$<$class T, class M$>$\newline
class v8\+::\+Persistent$<$ T, M $>$}

A \mbox{\hyperlink{classv8_1_1PersistentBase}{Persistent\+Base}} which allows copy and assignment.

Copy, assignment and destructor behavior is controlled by the traits class M.

CAVEAT\+: \mbox{\hyperlink{classv8_1_1Persistent}{Persistent}} objects do not have proper destruction behavior by default and as such will leak the object without explicit clear. Consider using {\ttfamily \mbox{\hyperlink{classv8_1_1Global}{v8\+::\+Global}}} instead which has proper destruction and move semantics.

\mbox{\hyperlink{classv8_1_1Persistent}{Persistent}} is a way to create a strong pointer from an off-\/heap object to another on-\/heap object. As long as the \mbox{\hyperlink{classv8_1_1Persistent}{Persistent}} handle is alive the GC will keep the object pointed to alive.

\doxysubsection{Detailed Description}
Can be used to avoid repeated expensive type checks for groups of objects that are expected to be similar (e.\+g. when Blink converts a bunch of Java\+Script objects to \char`\"{}\+Script\+Wrappable\char`\"{} after a \char`\"{}\+Has\+Instance\char`\"{} check) by making use of V8-\/internal \char`\"{}hidden classes\char`\"{}. An object that has passed the full check can be remembered via \{Update\}; further objects can be queried using \{Matches\}. Note that the answer will be conservative/\char`\"{}best-\/effort\char`\"{}\+: when \{Matches\} returns true, then the \{candidate\} can be relied upon to have the same shape/constructor/prototype/etc. as the \{baseline\}. Otherwise, no reliable statement can be made (the objects might still have indistinguishable shapes for all intents and purposes, but this mechanism, being optimized for speed, couldn\textquotesingle{}t determine that quickly).

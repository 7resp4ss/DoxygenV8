\doxysubsection{Detailed Description}
This class serves as a base class for recording event-\/based metrics in \mbox{\hyperlink{classv8_1_1V8}{V8}}. There a two kinds of metrics, those which are expected to be thread-\/safe and whose implementation is required to fulfill this requirement and those whose implementation does not have that requirement and only needs to be executable on the main thread. If such an event is triggered from a background thread, it will be delayed and executed by the foreground task runner.

The embedder is expected to call \mbox{\hyperlink{classv8_1_1Isolate_ae19bd6443174d5ed666132606d55d0be}{v8\+::\+Isolate\+::\+Set\+Metrics\+Recorder()}} providing its implementation and have the virtual methods overwritten for the events it cares about.

\doxysubsection{Detailed Description}
Compilation data that the embedder can cache and pass back to speed up future compilations. 

The data is produced if the Compiler\+Options passed to the compilation functions in \mbox{\hyperlink{classv8_1_1ScriptCompiler}{Script\+Compiler}} contains produce\+\_\+data\+\_\+to\+\_\+cache = true. The data to cache can then can be retrieved from \mbox{\hyperlink{classv8_1_1UnboundScript}{Unbound\+Script}}.

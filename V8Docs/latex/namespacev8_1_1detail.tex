\doxysubsection{Detailed Description}
Helper function template to create an array of fixed length, initialized by the provided initializer list, without explicitly specifying the array size, e.\+g.

auto arr = \mbox{\hyperlink{namespacev8_a225a36f8b8e3101e149b3b9adc050aa7}{v8\+::to\+\_\+array}}$<$Local$<$\+String$>$$>$(\{v8\+\_\+str(\char`\"{}one\char`\"{}), v8\+\_\+str(\char`\"{}two\char`\"{})\});

In the future, this may be replaced with or aliased to std\+::to\+\_\+array (under consideration for C++20).

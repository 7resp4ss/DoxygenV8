\doxysubsection{Detailed Description}
A move-\/only class for managing the lifetime of shared value conveyors used by \mbox{\hyperlink{classv8_1_1V8}{V8}} to keep JS shared values alive in transit when serialized. 

This class is not directly constructible and is always passed to a \mbox{\hyperlink{classv8_1_1ValueSerializer_1_1Delegate}{Value\+Serializer\+::\+Delegate}} via Value\+Serializer\+::\+Set\+Shared\+Value\+Conveyor.

The embedder must not destruct the \mbox{\hyperlink{classv8_1_1SharedValueConveyor}{Shared\+Value\+Conveyor}} until the associated serialized data will no longer be deserialized.

\doxysubsection{Detailed Description}
Returns a view onto a string\textquotesingle{}s contents.

WARNING\+: This does not copy the string\textquotesingle{}s contents, and will therefore be invalidated if the GC can move the string while the \mbox{\hyperlink{classv8_1_1String_1_1ValueView}{Value\+View}} is alive. It is therefore required that no GC or allocation can happen while there is an active \mbox{\hyperlink{classv8_1_1String_1_1ValueView}{Value\+View}}. This requirement may be relaxed in the future.

\mbox{\hyperlink{classv8_1_1V8}{V8}} strings are either encoded as one-\/byte or two-\/bytes per character.

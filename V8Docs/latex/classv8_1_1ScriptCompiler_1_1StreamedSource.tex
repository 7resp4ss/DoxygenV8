\doxysubsection{Detailed Description}
\mbox{\hyperlink{classv8_1_1ScriptCompiler_1_1Source}{Source}} code which can be streamed into \mbox{\hyperlink{classv8_1_1V8}{V8}} in pieces. 

It will be parsed while streaming and compiled after parsing has completed. \mbox{\hyperlink{classv8_1_1ScriptCompiler_1_1StreamedSource}{Streamed\+Source}} must be kept alive while the streaming task is run (see \mbox{\hyperlink{classv8_1_1ScriptCompiler_1_1ScriptStreamingTask}{Script\+Streaming\+Task}} below).

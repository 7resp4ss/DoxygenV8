{\footnotesize\ttfamily bool v8\+::base\+::\+Virtual\+Address\+Space\+Page\+Allocator\+::\+Decommit\+Pages (\begin{DoxyParamCaption}\item[{void $\ast$}]{address,  }\item[{size\+\_\+t}]{size }\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [override]}, {\ttfamily [virtual]}}



Decommits any wired memory pages in the given range, allowing the \mbox{\hyperlink{classv8_1_1base_1_1OS}{OS}} to reclaim them, and marks the region as inacessible (k\+No\+Access). 

The address range stays reserved and can be accessed again later by changing its permissions. However, in that case the memory content is guaranteed to be zero-\/initialized again. The memory must have been previously allocated by a call to Allocate\+Pages. Returns true on success, false otherwise.

Frees memory in the given \mbox{[}address, address + size) range. 

address and size should be operating system page-\/aligned. The next write to this memory area brings the memory transparently back. This should be treated as a hint to the \mbox{\hyperlink{classv8_1_1base_1_1OS}{OS}} that the pages are no longer needed. It does not guarantee that the pages will be discarded immediately or at all.

Performs embedder-\/specific operations to freeze the provided embedder object. The provided object {\itshape will} be frozen by Deep\+Freeze after this function returns, so only embedder-\/specific objects need to be frozen. This function {\itshape may not} create new JS objects or perform JS allocations. Any \mbox{\hyperlink{namespacev8}{v8}} objects reachable from the provided embedder object that should also be considered for freezing should be added to the children\+\_\+out parameter. Returns true if the operation completed successfully.

\doxysubsection{Detailed Description}
\mbox{\hyperlink{classv8_1_1CpuProfile}{Cpu\+Profile}} contains a CPU profile in a form of top-\/down call tree (from \mbox{\hyperlink{generate-bytecodes-builtins-list_8cc_ac0f2228420376f4db7e1274f2b41667c}{main()}} down to functions that do all the work).

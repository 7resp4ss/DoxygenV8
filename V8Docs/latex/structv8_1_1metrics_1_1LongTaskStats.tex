\doxysubsection{Detailed Description}
Experimental API intended for the Long\+Tasks UKM (crbug.\+com/1173527). 

The \mbox{\hyperlink{structv8_1_1metrics_1_1LongTaskStats_a18e223b0049fdd8237a5178965a6ac13}{Reset()}} method should be called at the start of a potential long task. The \mbox{\hyperlink{structv8_1_1metrics_1_1LongTaskStats_af6f26ac45fb0c25b7e04bd5137ad7521}{Get()}} method returns durations of \mbox{\hyperlink{classv8_1_1V8}{V8}} work that happened during the task.

This API is experimental and may be removed/changed in the future.

Decommits any wired memory pages in the given range, allowing the \mbox{\hyperlink{classv8_1_1base_1_1OS}{OS}} to reclaim them, and marks the region as inacessible (k\+No\+Access). The address range stays reserved and can be accessed again later by changing its permissions. However, in that case the memory content is guaranteed to be zero-\/initialized again. The memory must have been previously allocated by a call to Allocate\+Pages. Returns true on success, false otherwise.

\subsubsection*{template$<$typename T, typename = void$>$\newline
struct cppgc\+::\+Post\+Construction\+Callback\+Trait$<$ T, typename $>$}

Allows users to specify a post-\/construction callback for specific types. 

The callback is invoked on the instance of type \mbox{\hyperlink{classT}{T}} right after it has been constructed. This can be useful when the callback requires a fully-\/constructed object to be able to dispatch to virtual methods.

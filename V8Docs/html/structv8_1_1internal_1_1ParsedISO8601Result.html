Detailed Description

ParsedISO8601Result contains the parsed result of ISO 8601 grammar documented in #sec-temporal-iso8601grammar for TemporalInstantString, TemporalZonedDateTimeString, CalendarName, TemporalDateString, TemporalDateTimeString, TemporalMonthDayString, TemporalRelativeToString, TemporalTimeString, TimeZoneIdentifier, and TemporalYearMonthString.

For all the fields represented by int32_t, a special value kMinInt31 is used to represent the field is "undefined" after parsing.

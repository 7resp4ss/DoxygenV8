Atomic stop-the-world marking.

Incremental marking interleaves marking with the rest of the application workload on the same thread.

Specifies supported sweeping types.

Enumerator
kAtomic 

Atomic stop-the-world sweeping.

All of sweeping is performed at once.

kIncremental 

Incremental sweeping interleaves sweeping with the rest of the application workload on the same thread.

kIncrementalAndConcurrent 

Incremental and concurrent sweeping.

Sweeping is split and interleaved with the rest of the application.

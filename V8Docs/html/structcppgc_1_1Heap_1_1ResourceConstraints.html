size_t cppgc::Heap::ResourceConstraints::initial_heap_size_bytes = 0

Allows the heap to grow to some initial size in bytes before triggering garbage collections.

This is useful when it is known that applications need a certain minimum heap to run to avoid repeatedly invoking the garbage collector when growing the heap.

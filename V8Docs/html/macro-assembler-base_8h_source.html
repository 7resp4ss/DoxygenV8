77  // Changes the current sandboxing mode.

80  // their execution, such as JSEntry (which changes from unsandboxed to
81  // sandboxed execution mode). It should not be used by any other builtins.

187  ~HardAbortScope() { assembler_->set_abort_hard(old_value_); }

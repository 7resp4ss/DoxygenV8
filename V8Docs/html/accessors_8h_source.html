139  // Create an AccessorInfo. The setter is optional (can be nullptr).
140  //
141  // Note that the type of setter is AccessorNameBooleanSetterCallback instead

143  // set a (boolean) return value. The setter should roughly follow the same

145  // - The return value is unset iff there was an exception.

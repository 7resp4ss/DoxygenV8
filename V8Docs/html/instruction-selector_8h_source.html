553  // Used in pattern matching during code generation.
554  // Check if {node} can be covered while generating code for the current
555  // instruction. A node can be covered if the {user} of the node has the only
556  // edge, the two are in the same basic block, and there are no side-effects
557  // in-between. The last check is crucial for soundness.
558  // For pure nodes, CanCover(a,b) is checked to avoid duplicated execution:
559  // If this is not the case, code for b must still be generated for other
560  // users, and fusing is unlikely to improve performance.
561  bool CanCover(turboshaft::OpIndex user, turboshaft::OpIndex node) const;
562 
563  bool CanCoverProtectedLoad(turboshaft::OpIndex user,
564  turboshaft::OpIndex node) const;

566  // Used in pattern matching during code generation.
567  // This function checks that {node} and {user} are in the same basic block,
568  // and that {user} is the only user of {node} in this basic block. This
569  // check guarantees that there are no users of {node} scheduled between
570  // {node} and {user}, and thus we can select a single instruction for both
571  // nodes, if such an instruction exists. This check can be used for example
572  // when selecting instructions for:
573  // n = Int32Add(a, b)
574  // c = Word32Compare(n, 0, cond)
575  // Branch(c, true_label, false_label)
576  // Here we can generate a flag-setting add instruction, even if the add has
577  // uses in other basic blocks, since the flag-setting add instruction will
578  // still generate the result of the addition and not just set the flags.
579  // However, if we had uses of the add in the same basic block, we could have:
580  // n = Int32Add(a, b)
581  // o = OtherOp(n, ...)
582  // c = Word32Compare(n, 0, cond)
583  // Branch(c, true_label, false_label)
584  // where we cannot select the add and the compare together. If we were to
585  // select a flag-setting add instruction for Word32Compare and Int32Add while
586  // visiting Word32Compare, we would then have to select an instruction for
587  // OtherOp *afterwards*, which means we would attempt to use the result of
588  // the add before we have defined it.
589  bool IsOnlyUserOfNodeInSameBlock(turboshaft::OpIndex user,
590  turboshaft::OpIndex node) const;

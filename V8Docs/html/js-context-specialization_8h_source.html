29 // Specializes a given JSGraph to a given context, potentially constant folding
30 // some {LoadContextNoCell} nodes or strength reducing some {StoreContextNoCell}
31 // nodes. Additionally, constant-folds the function parameter if {closure} is
32 // given, and constant-folds import.meta loads if the corresponding object
33 // already exists.
34 //

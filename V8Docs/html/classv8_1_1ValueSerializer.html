Detailed Description

Value serialization compatible with the HTML structured clone algorithm.

The format is backward-compatible (i.e. safe to store to disk).

std::pair< uint8_t *, size_t > v8::ValueSerializer::Release ( )

Returns the stored data (allocated using the delegate's ReallocateBufferMemory) and its size.

This serializer should not be used once the buffer is released. The contents are undefined if a previous write has failed. Ownership of the buffer is transferred to the caller.

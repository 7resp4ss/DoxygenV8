76 // The CallFeedbackRelation provides the meaning of the call feedback for a
77 // TurboFan JSCall operator
78 // - kReceiver: The call target was Function.prototype.apply and its receiver
79 // was recorded as the feedback value.
80 // - kTarget: The call target was recorded as the feedback value.
81 // - kUnrelated: The feedback is no longer related to the call. If, during
82 // lowering, a JSCall (e.g. of a higher order function) is replaced by a
83 // JSCall with another target, the feedback has to be kept but is now
84 // unrelated.
85 enum class CallFeedbackRelation { kReceiver, kTarget, kUnrelated };
86
